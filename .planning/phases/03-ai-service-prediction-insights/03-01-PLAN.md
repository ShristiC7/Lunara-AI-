<action>
Create `ai/app/services/cycle_predictor.py`.
Implement `CyclePredictor` class:
- `predict`: Extracts cycle lengths from list. 
  - If 0 length: return default 28 days from current or last start.
  - If < 3: rolling average of cycle lengths. Confidence LOW.
  - If >= 3: Fit `LinearRegression` on index vs length. Confidence MEDIUM/HIGH.
  - Returns `predictedDate`, `ovulationWindow` (14 days before predicted date, ôÝ2 days).
</action>
<acceptance_criteria>
Logic correctly falls back to averages for small datasets and return correct confidence strings.
</acceptance_criteria>
